# rzuquim’s dotfiles

## Installation

> **Warning:** If you want to give these dotfiles a try, you should first fork this repository, review the code, and remove things you don’t want or need. Don’t blindly use my settings unless you know what that entails. Use at your own risk!

### Setup [POP_OS!](https://pop.system76.com/)

Open a terminal, paste this and hope for the best.

```
. <(curl -sL https://rzuquim.com/bootstrap_pop.sh)
```

> If the domain is not available use the raw file on the [GitHub](https://raw.githubusercontent.com/rzuquim/dotfiles/master/bootstrap/pop.sh) repository.
